# Board Work

이 디렉터리는 board 프로젝트의 dev 작업 진입 전 작업 후보, 작업 정책, 로드맵, 현재 작업 포인터, 작업 로그를 관리한다.

현재 문서는 골격이며, 실제 작업 단위 확정 전까지는 후보와 판단 근거를 분리해 기록한다.

## 문서 목록

- `work-candidates.md`: 아직 확정되지 않은 작업 후보를 관리한다.
- `work-policy.md`: 후보 검토, Roadmap 승격, dev 실행 기준을 관리한다.
- `work-roadmap.md`: 승인된 Phase / Cycle / Work Package / Task 구조를 관리한다.
- `work-index.md`: 현재 Phase / Cycle / Work Package / Focus Task 포인터를 관리한다.
- `work-log.md`: 실행 결과와 sync 판단을 기록한다.
