# SpecDrive

SpecDrive is a document-first AI collaboration workflow system.

This directory is organized so SpecDrive can later be separated into its own project or repository. It contains reusable documentation, templates, rules, configuration, scripts, and Codex skill assets.

Primary documentation starts at `specdrive/docs/README.md`.

Manual drafts are kept under `specdrive/manual/**`.

- `specdrive/manual/plan-manual.md`: `$plan` execution manual draft.
- `specdrive/manual/dev-manual.md`: `$dev` execution manual draft.
